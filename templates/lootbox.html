三、為什麼不是 1/機率 × 單價？

很多人會直覺算「100 抽才中一本，所以 1 本 = 5,000 元」，但這忽略了「過程中累積的星星」：

{% set demo_price = channels[0].price %} {% set main_prob = box.rewards|selectattr('name', 'equalto', box.main_prize_name)|first %} {% set draws_for_one = (1 / main_prob.prob)|round(0)|int %} {% set total_money = draws_for_one * demo_price %} {% set side_stars_per_draw = ev - main_prob.prob * box.main_prize_alt_value %} {% set side_stars_total = side_stars_per_draw * (draws_for_one - 1) %} {% set extra_books = side_stars_total / box.main_prize_alt_value %} {% set total_books = 1 + extra_books %} {% set net = total_money / total_books %} 
開 {{ draws_for_one }} 抽 ({{ "{:,}".format(total_money) }} 元):
  ┌─ 期望抽到 1 本 {{ box.main_prize_name }} (價值 {{ box.main_prize_alt_value }} 星)
  └─ 期望累積 {{ side_stars_total|round(0)|int }} 星 (從其他 {{ draws_for_one - 1 }} 個非中獎袋)
    ↓
  {{ side_stars_total|round(0)|int }} 星 / {{ box.main_prize_alt_value }} 星 = {{ extra_books|round(2) }} 本「換」來的書
    ↓
  {{ draws_for_one }} 抽總共 = 1 + {{ extra_books|round(2) }} = {{ total_books|round(2) }} 本書
    ↓
  每本平均 = {{ "{:,}".format(total_money) }} / {{ total_books|round(2) }} ≈ {{ "{:,}".format(net|round(0)|int) }} 元

⚠ 期望值的真實意義

本頁計算的是「長期 / 大樣本平均」成本 — 假設你抽夠多次，每本平均花費會收斂到上述數字。 單次運氣完全可能高於或低於：你可能 50 抽就中、也可能 200 抽都沒中。 EV 只反映機率設計的合理性下限，不保證個人結果。

💸 練功花錢前請評估自己負擔範圍。福袋本質是賭博性遊戲機制，預算之內當娛樂、超過就請停手。
